### Key Findings

- **19/19 routes functional**
- 256-byte payload limit enforced
- Dryrun mode works (hex returned, no broadcast)
- 30-min cooldown on /patch (configurable via PATCH_COOLDOWN_MINUTES env)
- Pagination on /read (limit/offset + total count)
- /split fixed — no more hardcoded testnet, no duplicate tx creation, no seed logging
- Seeds never stored or logged
- Admin routes (/admin, /admin-data, /housekeep) hidden from public docs
- GitHub repo synced: https://github.com/soosho/peerscript
