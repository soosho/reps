### Key Findings
- All 19 routes functional
- 256-byte payload limit enforced (257 rejected)
- Dryrun mode works (returns hex without broadcast)
- 30-min cooldown on /patch enforced (configurable via PATCH_COOLDOWN_MINUTES)
- Pagination works on /read (limit/offset with total count)
- /split returns 500 on testnet — needs debug
- Mempool conflicts return 502 instead of 409 — minor UX issue
- /tx and /block pass raw RPC errors (502) instead of 404 for invalid inputs
